🚀 THE FASTEST WAY TO START:

   1. Windows Users:
      → Double-click the file: START_JARVIS.bat
      → Wait for it to load
      → Browser opens automatically at http://localhost:3000
      → Done! Click the microphone and start talking
   
   2. Mac/Linux Users:
      → Open terminal
      → Run: chmod +x start_jarvis.sh && ./start_jarvis.sh
      → Wait for it to load
      → Open browser: http://localhost:3000
      → Click microphone and talk!


⏱️ IF THAT DOESN'T WORK (5-MINUTE MANUAL SETUP):

   Step 1 - Open First Terminal Window:
   ──────────────────────────────────
   cd path/to/JARVIS
   pip install -r requirements.txt
   python server.py
   
   Wait until you see: "🚀 JARVIS Server starting..."
   
   
   Step 2 - Open SECOND Terminal Window (keep first running!):
   ─────────────────────────────────────────────────────────
   cd path/to/JARVIS
   python agent.py
   
   Wait for agent to start...
   
   
   Step 3 - Open Web Browser:
   ────────────────────────
   Go to: http://localhost:3000
   
   
   Step 4 - Use JARVIS:

🆘 PROBLEMS?

   Issue: "Server Offline" message
   Fix: Make sure first terminal is running "python server.py"
   
   Issue: No voice response from JARVIS
   Fix: Make sure second terminal is running "python agent.py"
   
   Issue: Microphone not working
   Fix: Click on the page to enable AudioContext (browser requirement)
   
   Issue: "ModuleNotFoundError"
   Fix: Run: pip install -r requirements.txt
   
   Issue: Permission denied (Mac/Linux)
   Fix: Run: chmod +x start_jarvis.sh
